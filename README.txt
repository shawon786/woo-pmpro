Restrict products from being bought by non members in Paid Membership Pro 

== Description ==

This plugin will help you to restrict your marketplace's products from being bought by a non-memmer. Members will need to be registered via Paid Membership Pro plugin as a member. If you have a subscription based marketplace, then this plugin will help you to run your business smoothly.

> Compatible with popular Multivendor plugins- Dokan, WC Vendor, WC Marketplace. 

Want to earn from vendors as well as from customers via subscription? Then this plugin means a lot to you!
Edit the default category (and slug) to 'woo-pm' and see the magic! All the products under that category will now restricted! It will hide the add-to-cart button and force the visitor/customer to register via Paid Membership Pro. 

> Important Note- Once installed and activated, you will need to add a product category named- 'woo-pm' and the slug should be 'woo-pm'. 
Now all the products under this category will need PM Pro Membership to purchase!
And if your site is a multivendor one, then make this new category (woo-pm) the default category, so that all the products added by the vendors also goes under this category! Now all the visitors must have a membership to purchase the products under this category! 

Note: The register button will link the visitors to the default membership levels page (yourDomain.com/levels) 
So confirm that you are using the default levels page here: Dashboard-> Memberships -> Page Settings -> Pages -> Levels Page

[Read Complete Documentation](https://innovsit.com/docs/woo-product-restriction-for-pm-pro/how-to-install/)


Required Plugins:
1. WooCommerce 
2. Paid Membership Pro

Additional (For Multivendor):
Compatible with Dokan Multivendor plugin, WC Vendors, WC Marketplace plugins.

== Installation ==

### Direct Install From WordPress Dashboard

1. Go to Plugins page
2. Add New and search for 'Woo product restriction'
3. You will see the plugin Woo Product Restriction for Paid Membership Pro in the search list.
4. Install and activate the plugin. That's all.


### Manual Install From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin. A ZIP file will be downloaded.
2. Login to your site’s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click choose file, select the plugin file and click install

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the blue button above.A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favorite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin

* Once installed and activated, you will need to add a product category named- 'woo-pm' and the slug should be 'woo-pm'. 
Now all the products under this category will need PM Pro Membership to purchase!
And if your site is a multivendor one, then make this new category (woo-pm) the default category, so that all the products added by the vendors also goes under this category! Now all the visitors must have a membership to purchase the products under this category! 

Note: The register button will link the visitors to the default membership levels page (yourDomain.com/levels) 
So confirm that you are using the default levels page here: Dashboard-> Memberships -> Page Settings -> Pages -> Levels Page

No, this plugin does not need any additional settings.
